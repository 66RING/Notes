# break it down

拆解天才，分析元素，定向重塑

- genius
	* good memory
		+ encoding: 如比喻，顺口溜
	* a deep understanding
		+ processing: 建立信息间的联系，知识网, 结合实践
			+ feedback encoding
			+ height order learning: 分4档位
				1. priority/judgements, why? group this way or this way?
				2. **relationship and group**, 网络 -> inter mind map
				3. applied/related
				4. isolation, 最低, 如背诵，解释
